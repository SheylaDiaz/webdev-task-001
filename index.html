Web Development Vocabulary.

  • HTML: The standard markup language for creating web pages.
  • CSS: A stylesheet language used to describe the presentation of a document written in HTML.
  • JavaScript: A programming language that allows you to create dynamic and interactive content on web pages.
  • Tag: A code element used to mark up the start or end of an element in HTML.
  • Element: A component in an HTML document defined by tags and their content.
  • Dom: The in-memory tree representation of a web page.
  • Api: A contract for programs to talk to each other.

  • Async/Await : Syntactic sugar over Promises for sequential async code.
